# Dokumenty pro chatbota

Tato složka slouží k ukládání dokumentů, ze kterých chatbot čerpá data.

## Podporované formáty

- **PDF** (.pdf) - dokumenty, prezentace, materiály
- **Word** (.docx, .doc) - textové dokumenty
- **Text** (.txt) - prosté textové soubory
- **Odkazy** (.md) - markdown soubory s odkazy a poznámkami
- **Excel** (.xlsx, .xls) - tabulky s daty
- **Obrázky** (.jpg, .png) - vizuální materiály

## Struktura

Doporučujeme organizovat dokumenty do podsložek podle kategorií:

```
documents/
├── skolni_rad/          # školní řády, předpisy
├── ucebni_plany/        # učební plány, rozvrhy
├── informace_o_skole/   # obecné informace o škole
├── akce_a_udalosti/     # akce, události, oznámení
├── kontakty/            # kontaktní informace
├── jidelna/             # jídelníčky, informace o jídelně
└── ostatni/             # ostatní dokumenty
```

## Poznámky

- Všechny dokumenty by měly být v češtině
- Udržujte aktuální verze dokumentů
- Před nahráním zkontrolujte, že dokumenty neobsahují citlivé osobní údaje
- Používejte srozumitelné názvy souborů
